Getting Started

Welcome to StrophenBoost! Follow these steps to start streaming:

  1. Create an Account: Register as a broadcaster to access streaming features
  2. Create a Stream: Set up your stream with title and description
  3. Configure OBS: Use your RTMP URL and stream key in broadcasting software
  4. Go Live: Start streaming and share your stream with viewers
Pro Tip: Test your stream setup with a private stream before going live publicly.

Streaming Setup

OBS Studio Configuration

Configure OBS Studio with these settings:

  • Server: rtmp://your-domain.com:1935/live
  • Stream Key: Your unique stream key from the dashboard
  • Recommended Settings: 720p @ 30fps, 2500 kbps bitrate

Supported Formats

  • Output: HLS (.m3u8) and DASH (.mpd)
  • Video Codecs: H.264, H.265
  • Audio Codecs: AAC, MP3

Embed Integration

Embed your streams on any website using these options:

Iframe Embed

<iframe src="https://your-domain.com/embed/STREAM_ID" 
    width="800" height="450" frameborder="0" allowfullscreen>
</iframe>

JavaScript Embed

<div id="strophenboost-player"></div>
<script src="https://your-domain.com/embed.js"></script>

Responsive Embed

Use responsive containers for mobile-friendly embeds.

API Reference

StrophenBoost provides REST API endpoints for stream management:

Stream Operations

  • POST /create_stream - Create new stream
  • POST /start_stream/<id> - Start streaming
  • POST /stop_stream/<id> - Stop streaming
  • GET /stream_analytics/<id> - Get analytics
  • GET /embed_code/<id> - Get embed codes

Authentication

All API requests require valid session authentication.

Troubleshooting

Common Issues

Stream Won't Start
  • Check RTMP URL and stream key
  • Verify OBS connection settings
  • Check firewall and network settings
Poor Stream Quality
  • Reduce bitrate in OBS
  • Check internet connection speed
  • Lower resolution or framerate
Need Help? Contact our support team for assistance with technical issues.
